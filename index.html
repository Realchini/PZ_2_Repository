GIT - система керування версіями

Git – це розподілена система керування версіями, створена для керування проектами та відстеження змін у вихідному коді. Система Git дозволяє розробникам працювати паралельно, не заважаючи один одному, та надає потужні інструменти для злиття змін, зроблених різними розробниками. Тест
